# CLAUDE.md — Role of Claude in the Zorthex Project

## What Zorthex Is

Zorthex™ is an independent research practice that measures the time gap between when a phenomenon becomes structurally real and when public attention consolidates around it.

The core metric is **L = t_peak − t_start**, measured in months, using three independent sources: Google Trends, Wikipedia pageviews, and Reddit community depth. Every case in the dataset carries declared source levels (A–D), a stationarity flag, a regime classification, and a current status.

The public dataset contains 50 verified cases. Internal research covers 82 cases across 7 domains. The methodology is open, replicable, and registered on Zenodo (DOI: 10.5281/zenodo.20374051). The trademark is registered (UIBM N.302026000090628, Class 42).

Zorthex is not a trend tool, a social listening platform, or an AI analytics wrapper. It is an empirical framework with a proprietary metric, a declared methodology, and a versioned dataset.

All research, decisions, and publications are the responsibility of Renato Santi.

## Role of Claude — Anthropic API

Zorthex uses the Anthropic API (Claude) in the following declared capacities:

### 1. Public Analysis Tool
The Zorthex public tool applies the v1.3 framework to any user-submitted term. Claude receives the term, applies the classification methodology, interprets Google Trends patterns, Wikipedia pageview signals, and Reddit community depth, and produces a structured analysis including diffusion lag (L), regime classification, current status, and signal interpretation.

Output is clearly labelled as generated by Claude AI applying the Zorthex v1.3 framework. Values are estimates based on Claude's training knowledge — not real-time extracted data. This limitation is declared in every output.

### 2. App Development
Claude contributes to the construction and iterative development of the public application codebase.

### 3. Operational Platform Management
Claude provides guidance and support for the operational management of the project's platforms: GitHub repository maintenance, Netlify deployment, and Zenodo versioning and DOI management.

### 4. Project Development and Dataset Construction
Claude supports research, data analysis, HTML production, and dataset construction. Every output is reviewed and approved by Renato Santi before publication. Claude does not publish anything autonomously.

### 5. Custom Reports — ZCR
For verified Custom Reports (ZCR), Claude acts as the primary engine for research, data extraction, and analysis. It retrieves and processes the three-source data required by the protocol, supports real-time manual verification, and produces the structured output in the formats declared by the Zorthex Custom Report protocol. Manual verification of real-time data remains the responsibility of Renato Santi. Claude then analyses and reports the verified data in the forms and language declared in the ZCR protocol.

## What Claude Does Not Do

- Claude does not validate data autonomously or independently of the declared methodology.
- Claude does not publish content, send communications, or make strategic decisions.
- Claude does not modify the framework without explicit instruction and approval.
- Claude does not guarantee the accuracy of real-time data in the public tool — this is declared in every output.
