What is Zorthex
Zorthex is an independent research practice focused on institutional attention diffusion and infrastructure transitions.
The framework measures L = t_peak − t_start (months) — the observable gap between when a technology or infrastructure becomes operationally real and when public attention structurally consolidates around it.
Zorthex does not predict. It does not evaluate value, performance, or outcomes. It measures the temporal gap between operational emergence and institutional legibility for capital allocators. It is descriptive only and does not constitute investment advice.
Full methodology: zorthex.com/methodology.html

Framework v1.2 — Key Features
Core Definitions (locked across all versions)
t_start — first documented operational emergence, justified with primary source (Level A–D)
t_peak — first month of 12 consecutive months ≥25/100 on Google Trends Worldwide
L — t_peak − t_start in months
Classifications
STRUCTURAL — ≥25/100 for 12+ consecutive months (permanent once acquired)
OBSERVATION — above threshold, fewer than 12 consecutive months
BUBBLE — peaked above threshold, dropped below 25, no 12-month window (permanent)
4-Regime Diffusion Taxonomy (empirical, n=32)
Regime	Name	Pattern	Examples
1	Policy-Trigger	Slow ramp → regulatory breakout	PQC (91mo), Stablecoins (128mo)
2	Institutional Mass	Multi-year invisibility → explosive breakout	RWA (103mo)
3	Market-Narrative	Speculation cycles	Bitcoin (107mo), Crypto
4	Shock/Spoke	Single-event spike → memory-zero collapse	GDPR, NFT partial
Non-Stationarity Flag (mandatory per case)
STABLE — organic diffusion, L fully comparable cross-cluster
SEMI-STABLE — partial external influence, L usable with caution
NON-STATIONARY — exogenous shock dominant, excluded from L averaging
Layer Separation
Layer A — Hard Infrastructure (L-comparable): 18 cases
Layer B — Attention Phenomena (regime-classifiable only): 14 cases
